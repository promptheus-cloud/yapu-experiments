**Step 4: Update nginx for HMR WebSocket pass-through**

Edit the yapu-preview nginx config:
```bash

```

Add this location block BEFORE the existing `location /` block (the one with the bearer token check):

```nginx
    # HMR WebSocket pass-through — no bearer gate
    # Browser WebSocket API cannot send Authorization headers
    # Security: preview subdomain is already isolated from production
    location = /_next/webpack-hmr {
        proxy_pass http://127.0.0.1:3008;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        proxy_set_header Host $host;
        proxy_read_timeout 3600s;
        proxy_buffering off;
    }

    # Turbopack static assets for dev server (HMR chunks)
    location /_next/static {
        proxy_pass http://127.0.0.1:3008;
        proxy_set_header Host $host;
    }
```

Test and reload nginx:
