    **Step 2: Add CSS scroll animation keyframes to globals.css.** Open `app/globals.css` and add the following inside the existing `@theme inline { ... }` block, after the existing `--radius-4xl` line and before the closing `}`:

    ```css
    /* Logo carousel infinite scroll animations */
    --animate-scroll-logos: scroll-logos 35s linear infinite;
    --animate-scroll-logos-fast: scroll-logos 20s linear infinite;

    @keyframes scroll-logos {
      0%   { transform: translateX(0); }
      100% { transform: translateX(-50%); }
    }
    ```

    This creates `animate-scroll-logos` and `animate-scroll-logos-fast` Tailwind utilities. The `translateX(-50%)` assumes a doubled logo list (logos rendered twice side-by-side for seamless looping).

    Do NOT modify anything else in globals.css — only add the animation block inside `@theme inline`.
  </action>
