# Phase 4.2 Plan 01: ServiceModules Layout Rebuild and Homepage Text Fix Summary

Rebuilt ServiceModules from a 4-column card grid to 4 stacked full-width colored sections matching yapu.solutions. Updated hero text, CTA target, and all i18n section title strings across EN/FR/ES.

## What Was Built

### ServiceModules Component Rebuilt
The previous implementation was a 4-column card grid using shadcn Card components with icons. This bore no resemblance to the original yapu.solutions layout.

The new implementation renders 4 stacked `<section>` elements, each full-width with:
- `investor-services`: `bg-[rgb(217,217,217)]` (darker gray), text-brand title, text-cta link
- `data-insights`: `bg-gray-50`, text-brand title, text-cta link
- `digital-tools`: `bg-gray-50`, text-brand title, text-cta link
- `impact`: `bg-brand` (dark teal), text-white title/bullets, text-accent link

Each section renders: uppercase bold `<h2>` title, `<ul>` bullet list with circle markers, "read more" link.

### JSON Schema Updated (All 3 Locales)
`serviceModules` array schema changed: `description` and `icon` fields removed, `bullets: string[]` and `readMoreText: string` added.

Hero text corrected across all locales:
- EN: "YAPU facilitates resilience finance for the most vulnerable", CTA "try the APP now" → https://my.yapu.solutions
- FR: "YAPU facilite le financement de la resilience pour les plus vulnerables"
- ES: "YAPU facilita las finanzas de resiliencia para los mas vulnerables"

### Page.tsx Interface Updated
`ServiceModule` interface updated to match new JSON schema. `title` prop removed from `ServiceModules` JSX call. `servicesTitle` translation call removed.

### i18n Section Titles Corrected
All section titles updated to match yapu.solutions originals:
- EN: "Client voices" (testimonials), "Trusted by" (partners), "Our clients", "Stay up-to-date" (newsletter), "Subscribe!" (button)
- FR: "Voix des clients", "Nos partenaires de confiance", "Nos clients", "Restez a jour", "S'abonner!"
- ES: "Voces de clientes", "Socios de confianza", "Nuestros clientes", "Mantengase al dia", "Suscribirse!"

## Decisions Made

- **sectionConfig Record pattern**: Per-section styling isolated in a `Record<string, SectionStyle>` constant with a `defaultStyle` fallback. Clean separation from render logic.
- **ServiceModule interface change**: `bullets: string[]` + `readMoreText: string` replaces `description: string` + `icon: string`. This is a breaking schema change but all 3 locale JSON files were updated atomically.
- **ServiceModules removes title prop**: Matches original layout — no "Our Services" heading above sections.
- **Hero CTA changed to https://my.yapu.solutions**: Matches original yapu.solutions CTA target (external app link).
