**1. Create `components/CookieConsent.tsx`:**

A `'use client'` component that implements GDPR cookie consent with localStorage persistence.

Key implementation details:
- Import `useState`, `useEffect` from React, `useTranslations` from `next-intl`
- Use `useTranslations('CookieConsent')` for all UI strings
- `STORAGE_KEY = 'yapu_cookie_consent'`
- On mount (`useEffect`), read `localStorage.getItem(STORAGE_KEY)`. If value exists (`'granted'` or `'denied'`), do NOT show banner. If null, show banner.
- Initialize `showBanner` to `false` (server-safe default to avoid hydration mismatch). Set to `true` only inside `useEffect` when localStorage has no stored value.
- `accept()` function: stores `'granted'` in localStorage, hides banner, calls `window.gtag('consent', 'update', { analytics_storage: 'granted' })` if `window.gtag` exists
- `decline()` function: stores `'denied'` in localStorage, hides banner (GA4 stays denied by default)
- When `showBanner` is false, return `null`
- Banner UI: `fixed bottom-0 left-0 right-0 z-50 bg-brand text-white p-4 shadow-lg`
  - Inner: `max-w-6xl mx-auto flex flex-col md:flex-row items-center justify-between gap-4`
  - Text: `text-sm` with the translated message and a link to `https://yapu.solutions/privacy-policy` (target="_blank", rel="noopener noreferrer", className "underline ml-1")
  - Two buttons in a `flex gap-3`:
    - Decline: `px-4 py-2 rounded-full border border-white/40 text-sm hover:bg-white/10 transition-colors`
    - Accept: `px-4 py-2 rounded-full bg-accent text-brand text-sm font-semibold hover:opacity-90 transition-opacity`
  - Both buttons use translated text from `t('accept')` and `t('decline')`

Declare `gtag` on window for TypeScript:
```typescript
declare global {
  interface Window {
    gtag?: (...args: unknown[]) => void;
  }
}
```

**2. Create `components/GoogleAnalytics.tsx`:**

A server component (no `'use client'` directive) that renders GA4 scripts with Consent Mode v2 defaults.

Key implementation details:
- Import `Script` from `next/script`
- Read `const GA_ID = process.env.NEXT_PUBLIC_GA_MEASUREMENT_ID`
- If `!GA_ID`, return `null` (graceful degradation when env var not set)
- Return a React fragment with 3 `<Script>` tags:

Script 1 — Consent defaults (MUST load before GA4):
```jsx
<Script
  id="consent-defaults"
  strategy="beforeInteractive"
  dangerouslySetInnerHTML={{
    __html: `
      window.dataLayer = window.dataLayer || [];
      function gtag(){dataLayer.push(arguments);}
      gtag('consent', 'default', {
        'analytics_storage': 'denied',
        'ad_storage': 'denied',
        'ad_user_data': 'denied',
        'ad_personalization': 'denied',
        'wait_for_update': 500
      });
    `,
  }}
/>
```

Script 2 — GA4 library:
```jsx
<Script
  src={`https://www.googletagmanager.com/gtag/js?id=${GA_ID}`}
  strategy="afterInteractive"
/>
```

Script 3 — GA4 config:
```jsx
<Script
  id="ga4-init"
  strategy="afterInteractive"
  dangerouslySetInnerHTML={{
    __html: `
      window.dataLayer = window.dataLayer || [];
      function gtag(){dataLayer.push(arguments);}
      gtag('js', new Date());
      gtag('config', '${GA_ID}');
    `,
  }}
/>
```
